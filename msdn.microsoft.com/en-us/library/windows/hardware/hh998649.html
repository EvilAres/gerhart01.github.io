Windows Pen Hover Accuracy Test (System)

This manual test verifies a system's pen hover accuracy.

Test details

 

Associated requirements

System.Client.Digitizer.Pen.HoverAccuracy
System.Client.Digitizer.Touch.ColdBootLatency

See the system hardware requirements.

Platforms

Windows RT (ARM-based)
Windows 8 (x64)
Windows 8 (x86)
Windows RT 8.1
Windows 8.1 x64
Windows 8.1 x86

Expected run time

~2 minutes

Categories

Certification

Type

Manual

Running the test

Before you run the test, complete the test setup as described in the test requirements: System Client Testing Prerequisites and Digitizer Testing Prerequisites.

Troubleshooting

For troubleshooting information, see Troubleshooting System Client Testing and Troubleshooting Device Digitizer Testing.
