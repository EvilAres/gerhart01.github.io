Windows hardware certification: start here

Windows hardware requirements

To get started with certification review the Windows hardware design requirements for building exceptional hardware. When your Windows hardware is certified, it's designed to give your customers a great experience.

Windows hardware dashboard

Before you can start the Windows hardware certification process, you'll need to register with the hardware dashboard. It's where you'll go to submit your hardware for certification and manage it throughout the process.

Here's how to get registered:

  1. Establish a company
  2. Get a VeriSign certificate
  3. Add a user or migrate an account
  4. Sign legal agreements
  5. Sign in to the hardware dashboard
