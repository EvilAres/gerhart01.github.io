How to use the RA Tool to measure Touch Latency

|
English
None

About This Video

Learn the step by step techniques on how to use the Rotational Acoustic (RA) Tool to measure Touch Latency. This is a mechanical aid which was developed for the Windows 8 Touch Testing Labs in order to perform panning latency tests on different devices. Come away with the knowledge on how the RA tool is used to certify Windows 8 hardware.
